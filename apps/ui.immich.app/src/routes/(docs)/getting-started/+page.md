## Install

@immich/ui is published as an [npm](https://www.npmjs.com/package/@immich/ui) package. You can install it with your favorite package manager.

### pnpm

```
pnpm add @immich/ui
```

### npm

```
npm i @immich/ui
```

### yarn

```
yarn add @immich/ui
```
